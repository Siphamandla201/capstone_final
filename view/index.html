USERS

Methods Path Description
GET ('/users')

('/users/:UsersId) 		Getting all users

Getting a single user using an id
POST ('/register') Creating a new user
PUT ('/users/:UsersId') Updating an existing user details using id
DELETE ('/users/:UsersId') Deleting an existing user using an id
